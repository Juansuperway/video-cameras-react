# Proyecto_camaras
Proyecto para visualizador de cámaras - interfaz de usuario

Proyecto React sin Vite: sidebar de cámaras con interacción para cambiar la cámara activa y actualizar el cuadro de video.


Recordatorio del proyecto “Video Cámaras React”:

Proyecto React sin Vite, usando <script> y React desde CDN.

UI: Sidebar con lista de cámaras y área principal con cuadro de video.

Interacción: Click en cámara → cambia active, actualiza título y cuadro de video.

Archivos principales:

index.html → contiene todo el HTML y el React puro en <script>

styles.css → estilos de la app

Próximo paso: Agregar más funcionalidades, como mostrar video real o añadir nuevas cámaras dinámicamente.
